# AIと日本人 X投稿管理ツール

## 概要

「AIと日本人」および「善意あるAI社会」のX運用を効率化するために開発した個人用投稿管理ツールです。

当初は投稿候補の管理を目的としていましたが、運用を続ける中で、

* 過去投稿の再利用
* 再掲タイミング管理
* 投稿履歴管理

の重要性が高まり、現在は

**「X再掲管理ツール」**

として運用しています。

投稿の自動化は行わず、

* 投稿候補管理
* 投稿履歴管理
* 再掲候補抽出

までをシステムが担当し、

最終的な投稿判断は人間が行う設計としています。

---

## 主な機能

### 投稿管理

* 投稿一覧表示
* 投稿詳細表示
* 新規投稿登録
* 投稿編集
* 投稿削除

### 検索・絞り込み

* タイトル検索
* 本文検索
* カテゴリ絞り込み
* 投稿状態絞り込み
* 再掲候補抽出

### 再掲管理

* 投稿履歴管理
* 投稿回数管理
* 最終投稿日管理
* 再掲候補抽出

### 画像管理

* PNG → JPG変換
* WEBP → JPG変換
* JPEG正規化
* 画像プレビュー表示

### X連携

* X投稿画面起動
* 投稿済み登録
* 再掲履歴記録

## データベース

### posts

| 列名        |  内容   |
| ---------- | ------- |
| id         | 投稿ID  |
| title      | タイトル |
| category   | カテゴリ |
| content    | 本文   |
| image_path | 画像パス |
| post_count | 投稿回数 |
| created_at | 作成日時 |

### post_history

| 列名        |  内容   |
| ---------- | ------- |
| id        | 履歴ID   |
| post_id   | 投稿ID   |
| posted_at | 投稿日   |
| note      | 投稿メモ |

---

## 設計方針

* X APIは利用しない
* 自動投稿しない
* 最終投稿は人間が実施する
* 画像はJPEGへ統一保存
* 投稿履歴を蓄積する
* 半年後の自分が忘れないことを優先する
