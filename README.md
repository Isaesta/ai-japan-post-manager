# AIと日本人 X投稿管理ツール

## 概要

「AIと日本人」および「善意あるAI社会」のX運用を効率化するために開発した個人用投稿管理ツールです。

当初は投稿候補の管理を目的としていましたが、運用を続ける中で、

* 過去投稿の再利用
* 再掲タイミング管理
* 投稿履歴管理

の重要性が高まり、現在は

**「X再掲管理ツール」**

として運用しています。

投稿の自動化は行わず、

* 投稿候補管理
* 投稿履歴管理
* 再掲候補抽出
* 投稿資産管理

までをシステムが担当し、

最終的な投稿判断は人間が行う設計としています。

## 設計方針

* X APIは利用しない
* 自動投稿しない
* 最終投稿は人間が実施する
* 投稿履歴を蓄積する
* 投稿資産を再利用する
* 削除と非表示を分離する
* 画像はJPEGへ統一保存する
* 半年後の自分が忘れないことを優先する
